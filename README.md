## 5分钟学会MarkDown语法
参考：https://sspai.com/post/25137

学会MarkDown的好处：
- 从此不再烦恼排版问题，可以专注于文字本身
- 快速兼容GitHub，Jupyter Notebook，Kaggle，R语言社区等等（因为这就是程序员用的写作工具）
- 轻松转换成html，pdf，word
- 高级应用还可以兼容更多，如LaTeX
